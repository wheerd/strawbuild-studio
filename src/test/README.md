# Test IFC Files

This directory contains IFC (Industry Foundation Classes) test files used for import/export testing and validation.

## Files

- **AC20-FZK-Haus.ifc** - FZK House test model from the public FZK Building Models collection, commonly used for IFC compliance testing
- **DigitalHub_FM-ARC_v2.ifc** - Public IFC test file from [Open IFC Model](https://openifcmodel.cs.auckland.ac.nz/)
- **2022020320211122Wellness center Sama.ifc** - Public IFC test file from [Open IFC Model](https://openifcmodel.cs.auckland.ac.nz/)
- **Strawbale House.ifc** - Sample strawbale building model
- **strawbaler-export.ifc** - Export test fixture generated by Strawbaler
- **testsb.ifc** - Integration test fixture

## Purpose

These files are used to:

- Validate IFC import functionality across different IFC versions and authoring tools
- Test edge cases and complex geometry handling
- Ensure compatibility with real-world architectural models
- Verify export/import round-trip consistency
